* Some Key Binding

  | key         | Feature         |
  | Ctrl-a(N/I) | begin-of-line   |
  | Ctrl-e(N/I) | end-of-line     |
  | <SPC>w      | save-buffer     |
  | <SPC>q      | kill-buffer     |
  | <SPC>bm     | buffer-menu     |
  | <SPC>bb     | next-buffer     |
  | <SPC>bp     | previous-buffer |
